program 💥

🔈 ['prəʊɡræm]

🗝️ n. a plan of actions to be done or things to happen

🖼️ 在一间会议室里，公司团队正在进行项目规划。白板上写满了项目计划，包括时间表、任务清单和责任人。团队经理指着这些细节，解释每一步的具体安排，展示了'program'作为计划的含义。

🔍 想象'program'是一个行动计划或安排。无论是学校课程、计算机指令、广播节目，还是对机器或人的设置和训练，都可以看作是某种形式的'计划'。这个核心概念贯穿了'program'的各种用法，有助于你更好地理解和记忆它的多重含义。

💬 The teacher is explaining the educational program to the students.

词根词缀 📚

🌳 由词根 "gram"（写、记录）组成，前缀 "pro-"（向前，在前）表示提前写好的记录或计划。

🔗 1. grammar: 语法 2. diagram: 图表 3. telegram: 电报

💡 记忆 'program' 时，可以联想为 'pro'（提前）加上 'gram'（写），即提前写好的计划或安排，类似于在活动前准备好的步骤或指南。

衍生含义 🌱

  1. 🗝️ n. a set of instructions for a computer

    🖼️ 在一个明亮的办公室，程序员聚精会神地盯着电脑屏幕。他在编写一段代码，定义电脑程序的运作流程。屏幕上满是复杂的指令，展示了'program'作为一组计算机指令的含义。

    💬 He wrote a program to calculate the average score.

    ❓ 计算机执行的一系列指令也是一种行动计划

  2. 🗝️ n. a broadcast on television or radio

    🖼️ 在一个舒适的客厅中，家人们正围坐在电视机前收看他们最喜欢的电视剧。节目开始播放，背景音乐响起，大家都沉浸其中，展示了'program'作为电视或广播节目的含义。

    💬 What program are you watching on TV?

    ❓ 广播或电视节目是按计划安排的内容

  3. 🗝️ v. to set a machine to operate in a particular way

    🖼️ 在一个现代化的家庭厨房里，主人正在给洗衣机设定洗涤程序。他按下按钮，设置不同的参数，让洗衣机按预定的方式运行，展示了'program'作为设定机器操作方式的含义。

    💬 I programmed the washing machine to start at 7 am.

    ❓ 为机器设定操作计划

  4. 🗝️ v. to write computer programs

    🖼️ 在一个创意满满的科技初创公司，年轻的工程师正在电脑上编码，开发一款新的手机应用。他聚精会神地修复一个bug，展示了'program'作为编写计算机程序的动词含义。

    💬 She learned to program in Java last year.

    ❓ 编写计算机程序即是制定计算机的行动计划

  5. 🗝️ v. to train or influence to think or behave in a certain way

    🖼️ 在一个心理辅导中心，导师正通过一系列行为训练，帮助客户养成更健康的思维习惯。通过反复的练习，客户逐渐改变了他们的思维模式，展示了'program'作为影响或训练思维行为的含义。

    💬 Some people worry that TV programs children to be violent.

    ❓ 训练或影响他人的思维或行为也是一种计划实施
